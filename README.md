# FoxDial

Простое расширение Firefox в стиле Speed Dial: страница новой вкладки превращается в сетку плиток с любимыми сайтами.

## Возможности (MVP)

- Сетка плиток с фавиконами на странице новой вкладки
- Добавление, редактирование и удаление сайтов
- Открытие сайта по клику на плитку
- Данные хранятся локально (`browser.storage.local`)

## Структура

| Файл | Назначение |
|------|------------|
| `manifest.json` | Манифест расширения (MV3), переопределяет `newtab` |
| `newtab.html` | Разметка страницы новой вкладки и диалога |
| `newtab.css` | Стили |
| `newtab.js` | Логика: хранилище, рендер, добавление/редактирование/удаление |
| `icons/` | Иконки расширения |

## Запуск (временная установка)

1. Открыть `about:debugging#/runtime/this-firefox`
2. Нажать **Load Temporary Add-on…**
3. Выбрать файл `manifest.json` в этой папке
4. Открыть новую вкладку

Временное расширение живёт до перезапуска Firefox.

## Сборка для публикации

```sh
zip -r -FS foxdial.zip manifest.json newtab.html newtab.css newtab.js icons/
```

Получившийся `foxdial.zip` можно загрузить на [addons.mozilla.org](https://addons.mozilla.org/developers/).
